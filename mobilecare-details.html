Project Description

This ASP.NET Core MVC web application is designed specifically for the healthcare industry. It enables careworkers to log in, view bookings, and document reports on healthcare activities for individual patients.

Patients can log in to access their reports, while administrators have the capability to generate invoices for each patient booking.
